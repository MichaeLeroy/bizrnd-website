exec as an alternative to import and reload

# Use exec to avoid the pitfalls of import and reload when developing
# modules using the interpreter

  exec(open('my_module.py').read())

Note that using exec in this manner bypasses the namespace encapsulation that one gets when using the more conventional import and reload paradigm.
