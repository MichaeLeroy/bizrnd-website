Python operators presented in order of increasing precedence; later entries bind more tightly than the earlier ones. Sources are: the language reference and Learning Python. This table was created for the 3.x branch of Python, but is probably largely applicable to 2.x Python.

Operator Description Associativity
lambda Anonymous Function Definition Left to Right
if else Ternary Conditional Operator Left to Right, Short Circuits
or Logical Or Left to Right, Short Circuits
and Logical And Left to Right, Short Circuits
not Logical Negation Unary, Left to Right
in, not in ◾ is, is not ◾ <, <=, >, >=, !=, == Membership ◾ Identity ◾ Logical Comparisons Chained Left to Right
| Bitwise Or and Set Union Left to Right
^ Bitwise XOr and Set Exclusive Disjunction Left to Right
& Bitwise And and Set Intersection Left to Right
<<, >> Bitshifts Left to Right
+ ◾ - Addition and Concatenation ◾ Subtraction and Set Difference Left to Right
* ◾ / ◾ // ◾ % Multiplication and Repeat ◾ True Division ◾ Floor Division ◾ Remainder and Format Left to Right
+ ◾ - ◾ ~ Positive ◾ Negative ◾ Bitwise Not Unary, Left to Right
** Exponentiation Right to Left (but less bound than a unary +, - or ~ to its immediate right)
[] ◾ [:] ◾ () ◾ . Subscription ◾ Slice ◾ Call ◾ Attribute Reference Left to Right
() ◾ [] ◾ {} Binding, Tuple Display and Generator ◾ List Display and Comprehension ◾ Dictionary and Set Display and Comprehension Left to Right
